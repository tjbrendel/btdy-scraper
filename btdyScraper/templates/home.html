About

This site acts as the points system of the BTDY iRacing league. Both series points will be stored here and should be considered the source of truth for anything points related.

Features

Season Standings

Under each series menu item you can view the season standings of the most current season. Points are calculated according to the rules layed out in the official BTDY rules documents:

  • BTDY Contender Rules
  • BTDY Premier Rules

Penalty Report

Under each series menu there is a link to the penalty report. This report will give information on all the current penalties issued across the season. Information like round number, track, driver name, and reason are displayed here.

Race Sessions

When navigating through a series, on the left hand side, will be the list of completed races for the season. You can click on any of those to see the results for that session. Information on these pages include high level information on the race like number of laps, cautions, etc. as well as information on each driver.

For each driver you will see some basic information like their start/finish position, number of incidents, etc. You will also see all the total points that the driver has earned for that race and a breakdown of that toal points. This is all same information that has been presented from previous seasons.

Drop Weeks

After 10 weeks drop weeks will start being applied to season results. For Contender two drop weeks will be applied and for Premier one will be per the rules. A drivers lowest total point race will be considered for drop week(s). A footnote will show after that 10 week period so at any point you can click on it to see which races are currently being dropped.

Admin Features

Adding race data

When logging in you will be given an option to add a recently completed race. Filling out the form will pull the data from the iRacing service, perform the calculations based on the rules and selected bonuses, and store the data. Once complete you will be presented with sessions results for that race.

Modifying Race Results

On the session results page for any given race you can click on the name of the individual driver. This will take you to an edit screen where you can modify things like total points earned, payout ammounts, issue penalties, deleting records, and add notes. Issuing penalties will automatically populate the penalty report for that series.

Deleting Race Results

If something gets messed up with the whole session and you need to restart you can clear out the entire race session by clicking the delete race button. You can then re-pull the data with the Add Race form.
